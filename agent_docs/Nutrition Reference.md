## Common Tracking Mistakes

### Forgotten Items (Track These!)
- Club Mate: 100 cal
- Beers: 210 cal each
- Cooking fats: 45 cal/tsp
- Snacking: 100-200 cal
- Yogurt drinks: 136 cal per half
- **Total impact if untracked:** 200-500 cal/day

### Low-Protein Traps ❌
- Schupfnudeln + Sauerkraut: Only 6g protein for 310 cal
- Weißwurst + Pretzel: Only 18g protein for 500 cal
- Gnocchi without protein: Only 8g protein
- Pizza without meat: Only 12g per slice

### High-Protein Strategies ✅
- **Always pair carbs with protein source**
- **Double-scoop shakes when dinner is light**
- **Smoked fish at breakfast = 15-20g head start**
- **Chicken dürüm for lunch = 50g (half daily target)**
- **Veggie mince in rice bowls = 71g protein bomb**
